# TON Web Page Clone

This project is a **pixel-perfect front-end clone** of the "TON Web Page" UI design. 
The goal was to replicate the design as accurately as possible using **HTML**, 
**CSS**, and **Bootstrap**, with full responsiveness across desktop and mobile devices.

## 🚀 Features

* Fully responsive layout using Bootstrap 5
* Clean and semantic HTML5 structure
* Custom CSS for pixel-perfect styling
* Card-based training section with overlay text
* Insight section with call-to-action
* Footer with branding and media

## 💻 How to Run the Project

### Option 1: Open Locally

1. **Clone or download** this repository.
2. Ensure all image assets are available in the `images/` folder.
3. Open `index.html` in any modern web browser.

### Option 2: Using Live Server (Recommended)

1. Install the **Live Server extension** in VS Code.
2. Right-click on `index.html` and choose **"Open with Live Server"**.
3. The page will launch in your default browser with live reload support.
